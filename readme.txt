The Tide Graph Wordpress plugin shows a graph of the water levels in various areas in the US.

== Description ==

The popular iOS Tide Graph app is now available as a Wordpress widget.

== Installation ==

To install Tide Graph,

1. Upload `tide-graph.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Appearance->Widgets and enable the Tide Graph widget.
4. To change the settings, go to Appearance->Customize->Widgets. Then select on the area where your widget (ex. Footer). Finally select Tide Graph Widget to see the options: Background, Scale, Location, Latitude, and Longitude.


== Frequently Asked Questions ==

= How do I change the location =

Click on the map button and select a new location. Then highlight the name of the station that appears above the graph.

= What colors are allowed =

Colors can be white, black, transparent, or a 6 digit hex value preceeded by a # sign (ex. #FFFFFF)
